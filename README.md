Cosmic Jump

Cosmic Jump is a 2D arcade survival game developed in C++ using the Raylib library. Navigate your Raven through the cosmic lovecraftian-like void, avoid obstacles, and beware of the watching Kraken eye in the background.

Features a classic high-score chasing loop and a Local Multiplayer mode to challenge a friend.

Features

Single Player Mode: Test your reflexes and beat your high score.

2-Player Local Co-op/Versus: Play with a friend on the same keyboard.

Parallax Backgrounds: Immersive depth with multiple scrolling layers.

Audio: Retro soundtracks and pixel-perfect sound effects.

UI: Custom pixel fonts, clickable menus, and intuitive navigation.

Polish: Includes Pause menu, Mute functionality, and a Tutorial overlay.

Controls

Action

Player 1 (White Raven)

Player 2 (Red Raven)

Jump / Fly SPACE | UP ARROW
Pause Game | ESCAPE

Technologies Used

Language: C++

Library: Raylib 5.0

IDE: Visual Studio 2022

How to Build

Clone this repository.

Open the solution file (.sln) in Visual Studio 2022.

Ensure the Configuration is set to Debug or Release.

Build the solution.
